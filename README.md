# socket-io-chat

## Configuration files

You need to create two configuration files before running `npm run start`:

### Server config

Path: `/config.json`.

```
{
    "address": "127.0.0.1",
    "port": 8080
}
```

### Client config

Path: `/src/config.json`.

```
{
    "server": "127.0.0.1",
    "port": 8080
}
```
